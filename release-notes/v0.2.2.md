# Release v0.2.2

Data: 2026-04-09

Rilasciata la versione `v0.2.2`.
Questa release include miglioramenti alla pipeline CI/build, correzioni importanti in CoreJS e diverse nuove migliorie al motore/solver e all'integrazione Tauri.

## Highlights

- Aggiunto supporto per il motore "Popeye (ASM)".
- Miglioramenti al Tauri bridge e al solver Rust: supporto per solution e threat data nelle message flow.
- Stato reattivo (`signals`) per alcune parti di `DbmanagerService`.
- Tracciamento del tempo di esecuzione del solver e aggiornamenti alle opzioni di refutations (conteggio/visualizzazione).

## Fix principali

- Fix: pawn pushes rispettano le caselle occupate (CoreJS).
- Fix: evitare mutazione di `engineConfigurationsByEngine` in `Problem.toJson`.
- Fix vari per `problem-solver` (honor effective `refutations_try` in streaming).
- Correzioni e rifiniture di lint e test.

## CI / Build

- Aggiornata la pipeline CI: ordine dei job, rimozione di step ridondanti, installazione dipendenze workspace e build `CoreJS` in CI.
- Script di build aggiornati per usare `pnpm` nella compilazione TypeScript.

## Altri cambiamenti

- Normalizzazione dei manifest e preparazione della release (`chore(release): prepare v0.2.2`).
- Aggiornamento dipendenze e migliorie di refactor (Vitest, test, ecc.).
